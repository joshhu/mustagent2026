本課程部分檔案較大，使用了 git-lfs，請先安裝 git-lfs，再使用 https 方法來 clone 本課程，無法使用 zip 下載。

```bash
git lfs install
git clone https://github.com/joshhu/mustagent2026.git
```

## 課程簡介

本課程涵蓋 AI Agent 智慧體的完整知識體系，從 Vibe Coding 理論基礎、AI 開發工具實戰、MCP 協定與工作流程，到 Claude Code 深度應用與 Spec-Driven Development 開發方法論，帶領學生掌握 AI Agent 時代的軟體開發新範式。

## 課程目標

- 了解 AI Agent 的發展歷程與核心概念
- 掌握 Vibe Coding 與意圖導向開發的理論基礎
- 熟練使用 AI 開發工具（Antigravity、Claude Code 等）
- 理解 MCP 協定、Rules 與 Workflows 的設計
- 掌握 Claude Code 的運作原理、功能與記憶系統
- 學會 Harness Engineering 與 Spec-Driven Development 方法論
- 具備使用 GitHub Spec Kit 進行專案開發的能力
